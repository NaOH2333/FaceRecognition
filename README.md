# FaceRecognition人脸识别小程序
##介绍
使用 PolarDB 云数据库、云服务器与百度 AI 开放平台的人脸识别 api 开发，该小程序通过特别的人脸标识码来标记人脸，已有体验版小程序可使用

基于java11开发，java8已测试 完美运行，java8以下不保证项目的完美运行
