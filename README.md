# FaceRecognition人脸识别小程序
## 介绍
基于百度人脸识别API的人脸识别信息收集小程序，使用 PolarDB 云数据库、云服务器搭建开发；该小程序通过特别的人脸标识码来标记人脸，微信调用gps记录地理位置信息，并记录用户相关信息。
已有体验版小程序已通过微信小程序审核，正在申请域名，准备正式上线微信小程序。
