# FaceRecognition人脸识别小程序
## 介绍
使用 PolarDB 云数据库、云服务器与百度 AI 开放平台的人脸识别 api 开发，该小程序通过特别的人脸标识码来标记人脸，已有体验版小程序已通过微信小程序审核；
正在申请域名，准备正式上线微信小程序
